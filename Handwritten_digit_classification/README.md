# Digit Classification using Neural Networks

## Overview
This project performs handwritten digit classification using a neural network model.  
The dataset consists of numerical digit samples with labeled classes from 0 to 9.

## Dataset
- Digits dataset (scikit-learn)

## Approach

- Multiclass classification
