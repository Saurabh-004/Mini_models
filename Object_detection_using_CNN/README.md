# Image Classification using CNN on CIFAR-10

## Overview
This project implements a Convolutional Neural Network (CNN) to classify images from the CIFAR-10 dataset.  
The dataset contains small color images belonging to 10 different object categories.

## Dataset
- Loaded using `datasets.cifar10.load_data()`
- Classes include airplanes, automobiles, birds, cats, deer, dogs, frogs, horses, ships, and trucks

- Model training and evaluation
